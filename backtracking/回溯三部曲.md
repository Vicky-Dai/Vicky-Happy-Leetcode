void backtracking(参数)

2. 回溯终止条件
if (终止条件) {
    存放结果;
    return;
}
3. 回溯搜索的遍历过程
for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
    处理节点;
    backtracking(路径，选择列表); // 递归
    回溯，撤销处理结果
}

整合如下，这份模板很重要，后面做回溯法的题目都靠它了！
void backtracking(参数) {
    if (终止条件) {
        存放结果;
        return;
    }

    for (选择：本层集合中元素（树中节点孩子的数量就是集合的大小）) {
        处理节点;
        backtracking(路径，选择列表); // 递归
        回溯，撤销处理结果
    }
}
