# 动态规划问题类型总结

## 概述
动态规划是一种通过将复杂问题分解为更简单的子问题来解决优化问题的算法设计技术。不同类型的DP适用于解决不同领域的问题。

---

## 1. 线性DP - 解决序列优化问题

### 核心问题类型：
- **最优化问题**：在给定约束下找到最优解
- **计数问题**：计算满足条件的方案数
- **决策问题**：每一步都有选择，需要做出最优决策

### 具体应用：
```python
# 斐波那契数列 - 计算问题
# 问题：计算第n个斐波那契数
# 解决：避免重复计算，用DP存储中间结果
# 应用：金融建模、生物种群增长、算法复杂度分析

# 爬楼梯 - 计数问题  
# 问题：有多少种方法爬到第n阶
# 解决：将大问题分解为子问题
# 应用：路径规划、组合数学、概率计算

# 最小花费爬楼梯 - 最优化问题
# 问题：找到爬到楼顶的最小花费
# 解决：每一步选择最优策略
# 应用：成本优化、资源分配、决策分析
```

### 典型特征：
- 一维数组存储状态
- 状态转移相对简单
- 通常从前往后遍历
- 时间复杂度通常为O(n)

### 应用场景：
- 金融投资决策
- 路径规划算法
- 资源调度优化
- 概率统计计算

---

## 2. 背包DP - 解决资源分配问题

### 核心问题类型：
- **资源约束优化**：在有限资源下获得最大价值
- **选择问题**：从多个选项中选择最优组合
- **容量限制问题**：在容量限制下做出最优选择

### 具体应用：
```python
# 01背包 - 不可重复选择
# 问题：每个物品只能选一次，在容量限制下获得最大价值
# 解决：dp[j] = max(dp[j], dp[j-weight[i]] + value[i])
# 应用：投资组合选择、装载问题、资源分配

# 完全背包 - 可重复选择
# 问题：每个物品可以选多次，在容量限制下获得最大价值
# 解决：允许重复使用，从前往后遍历
# 应用：货币兑换、无限资源分配、重复投资

# 多重背包 - 有限次数选择
# 问题：每个物品有数量限制，在容量限制下获得最大价值
# 解决：在01背包基础上增加数量循环
# 应用：库存管理、有限资源分配、批量采购
```

### 典型特征：
- 容量约束和物品选择
- 遍历顺序很重要（01背包从后往前，完全背包从前往后）
- 状态转移涉及选择和不选择
- 时间复杂度通常为O(n×capacity)

### 应用场景：
- 投资组合优化
- 资源调度系统
- 装载运输问题
- 预算分配决策

---

## 3. 子序列DP - 解决字符串/数组匹配问题

### 核心问题类型：
- **序列匹配**：找两个序列的公共部分
- **序列优化**：找满足条件的最优子序列
- **编辑操作**：计算将一个序列转换为另一个序列的最小操作数

### 具体应用：
```python
# LIS (最长递增子序列) - 序列优化
# 问题：找到数组中最长的递增子序列
# 解决：dp[i]表示以nums[i]结尾的最长递增子序列长度
# 应用：股票价格分析、DNA序列分析、任务调度

# LCS (最长公共子序列) - 序列匹配
# 问题：找到两个字符串的最长公共子序列
# 解决：dp[i][j]表示text1[0:i]和text2[0:j]的LCS长度
# 应用：文本相似度、版本控制、生物信息学

# 编辑距离 - 编辑操作
# 问题：计算将一个字符串转换为另一个字符串的最小操作数
# 解决：dp[i][j]表示word1[0:i]转换为word2[0:j]的最小操作数
# 应用：拼写检查、文本纠错、模糊匹配

# 最大子数组和 - 连续子序列优化
# 问题：找到数组中和最大的连续子数组
# 解决：dp[i]表示以nums[i]结尾的最大子数组和
# 应用：信号处理、金融分析、数据挖掘
```

### 典型特征：
- 二维数组存储状态
- 需要两层循环比较
- 状态转移涉及字符/元素比较
- 时间复杂度通常为O(m×n)

### 应用场景：
- 文本处理和自然语言处理
- 生物信息学和基因分析
- 数据挖掘和模式识别
- 版本控制和差异比较

---

## 4. 矩阵DP - 解决二维路径问题

### 核心问题类型：
- **路径计数**：计算从起点到终点的路径数
- **路径优化**：找到从起点到终点的最优路径
- **区域优化**：在二维区域中找到满足条件的最优子区域

### 具体应用：
```python
# 不同路径 - 路径计数
# 问题：从左上角到右下角有多少种不同的路径
# 解决：dp[i][j]表示从(0,0)到(i,j)的路径数
# 应用：机器人路径规划、游戏AI、网络路由

# 最大正方形 - 区域优化
# 问题：在矩阵中找到最大的全1正方形
# 解决：dp[i][j]表示以(i,j)为右下角的最大正方形边长
# 应用：图像处理、计算机视觉、模式识别

# 最小路径和 - 路径优化
# 问题：从左上角到右下角的最小路径和
# 解决：dp[i][j]表示从(0,0)到(i,j)的最小路径和
# 应用：地图导航、成本优化、资源分配
```

### 典型特征：
- 二维数组存储状态
- 需要考虑边界条件
- 状态转移涉及上下左右方向
- 时间复杂度通常为O(m×n)

### 应用场景：
- 图像处理和计算机视觉
- 游戏开发和AI算法
- 地图导航和路径规划
- 网络优化和路由算法

---

## 5. 特殊DP - 解决特定领域问题

### 核心问题类型：
- **状态机问题**：系统有多个状态，状态间有转换关系
- **约束优化**：在特定约束下找到最优解
- **决策序列**：一系列相关决策的最优化

### 具体应用：
```python
# 打家劫舍 - 约束优化
# 问题：不能偷相邻房屋，求最大收益
# 解决：dp[i]表示偷到第i个房屋时的最大金额
# 应用：资源分配、约束优化、决策分析

# 买卖股票 - 状态机问题
# 问题：在股票价格变化中通过买卖获得最大收益
# 解决：用状态机表示持有/不持有股票的状态
# 应用：金融交易、投资策略、风险管理
```

### 典型特征：
- 多状态表示
- 状态间有转换关系
- 需要考虑特定约束
- 时间复杂度通常为O(n×k)，k为状态数

### 应用场景：
- 金融建模和交易策略
- 系统控制和状态管理
- 决策分析和优化
- 自动化和智能系统

---

## 总结对比表

| DP类型 | 主要解决问题 | 典型特征 | 应用场景 | 时间复杂度 |
|--------|-------------|----------|----------|------------|
| **线性DP** | 序列优化、计数、决策 | 一维数组，状态转移简单 | 金融建模、路径规划、资源调度 | O(n) |
| **背包DP** | 资源分配、选择优化 | 容量约束，物品选择 | 投资组合、资源调度、装载问题 | O(n×capacity) |
| **子序列DP** | 字符串匹配、序列优化 | 二维数组，序列比较 | 文本处理、生物信息学、编辑距离 | O(m×n) |
| **矩阵DP** | 二维路径、区域优化 | 二维数组，路径规划 | 图像处理、游戏AI、路径规划 | O(m×n) |
| **特殊DP** | 状态机、约束优化 | 多状态、特定约束 | 金融建模、系统控制、决策优化 | O(n×k) |

---

## 选择DP类型的判断标准

### 1. 问题特征分析
- **一维序列问题** → 线性DP
- **资源分配问题** → 背包DP
- **字符串/数组匹配** → 子序列DP
- **二维路径问题** → 矩阵DP
- **多状态转换问题** → 特殊DP

### 2. 状态定义模式
- **dp[i]** → 线性DP或特殊DP
- **dp[i][j]** → 子序列DP或矩阵DP
- **dp[j]** → 背包DP（一维优化）

### 3. 状态转移特征
- **简单递推** → 线性DP
- **选择/不选择** → 背包DP
- **字符/元素比较** → 子序列DP
- **方向性转移** → 矩阵DP
- **状态转换** → 特殊DP

### 4. 约束条件
- **无特殊约束** → 线性DP
- **容量约束** → 背包DP
- **序列约束** → 子序列DP
- **路径约束** → 矩阵DP
- **状态约束** → 特殊DP

## 实际应用建议

### 1. 问题识别流程
1. 分析问题是否具有最优子结构
2. 判断是否存在重叠子问题
3. 确定问题的维度（一维/二维）
4. 分析约束条件和选择特征
5. 选择合适的DP类型

### 2. 解题步骤
1. 定义状态：dp数组的含义
2. 状态转移：递推公式
3. 初始化：边界条件
4. 遍历顺序：确保状态依赖
5. 空间优化：状态压缩

### 3. 常见陷阱
- 遍历顺序错误（特别是背包问题）
- 边界条件处理不当
- 状态定义不清晰
- 空间优化过度导致逻辑错误

通过理解这些DP类型及其解决的问题，可以更快速地识别题目类型并选择合适的解题方法！
